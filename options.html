EasyNotes

User Manual

Overview

The purpose of this app is to create a space for users to quickly jot down any notes and information they need to save. To use the app, it's very simple. Whenever you write something, the app allows you to save it in 3 different slots. These slots allow you to retrieve and store information with the corresponding buttons for save and load.

How to Save

Choose the slot where you want to save; for example, below we want to save in slot 1. After inputting the text and title, click the Save 1 button.

How to Load

Choose the slot you want to load from; in the example below, we want to load what we saved in slot 1. Click the Load 1 button.
